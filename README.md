# 🧙 Wizardwerks Guessing Game

A web-based number guessing game built with Python and Streamlit.  
Developed as part of my Python learning journey at Wizardwerks Enterprise Labs.

## 🎮 Play the Game

👉 [Click here to play](https://wizardwerks-python-production.up.railway.app)

## About

The player guesses a randomly generated number between 1 and 10.  
The game tracks attempts, provides high/low hints, and handles invalid input gracefully.

## Built With

- Python 3.14
- Streamlit
- Custom CSS styling with Wizardwerks branding

## Concepts Demonstrated

- Variables and data types
- Conditionals and loops
- Error handling with try/except
- Session state management
- Web app deployment
